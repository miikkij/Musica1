Composer v2 Features — What matters most?

Select the features you consider highest priority. We'll design them in order.

1

Clip Looping / Repeat

Repeat a clip N times on the timeline so a 15s drum loop fills 2 minutes. Visual "grow loop" handle on clip edges.

2

Zoom & Scroll

Zoom in/out on the timeline (Ctrl+scroll or buttons), horizontal scrollbar, minimap overview. See minutes of audio at once or zoom into bar-level detail.

3

Clip Editing on Timeline

Move clips by dragging, resize/trim clip edges, delete clips, duplicate clips. Right-click context menu.

4

Enhanced Track Controls

Rename tracks, reorder tracks (drag up/down), per-track color, delete track, add empty track button. Track height adjustment.

5

Longer Compositions

Generate 8/16/32 bar clips, place multiple clips on the same track at different positions. Build a 3-5 minute song from sections (intro, verse, chorus, bridge, outro).

6

Playback Region / Loop Region

Select a region on the timeline to loop playback (e.g., loop bars 5-12). Visual markers for loop start/end. Click-to-seek on timeline ruler.

7

Per-Track Waveform Colors

Each track gets its own color (drums=red, bass=blue, melody=green). Color picker in track header. Makes it easy to visually distinguish instruments.
