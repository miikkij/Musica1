Mode Behaviors

▶ Cursor Mode
  • Click timeline → seek playhead
  • Click time ruler → seek
  • Drag on ruler → set loop region
  • Default mode
↔ Move Mode
  • Drag clip → move left/right
  • Snaps to bar grid
  • Delete key → remove clip
  • Right-click → context menu
[ ] Select Mode
  • Drag → select time region
  • Selected region = loop region
  • Loop checkbox → loop selection
  • Shift+click → extend selection
